# Tech Stack

## Core Framework
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Shadcn UI](https://ui.shadcn.com/)

## State & Data Management
- **API State:** [TanStack Query](https://tanstack.com/query/latest) (React Query) - For robust data fetching, caching, and synchronization with the existing backend.
- **Global UI State:** [Zustand](https://docs.pmnd.rs/zustand/getting-started/introduction) - A lightweight, performant solution for managing global dashboard state and user preferences.
- **Form Management:** [React Hook Form](https://react-hook-form.com/) - For performant and flexible form handling in budget and client management screens.
- **Validation:** [Zod](https://zod.dev/) - For schema-based validation of form data and API responses.

## Development Tools
- **Icons:** [Lucide React](https://lucide.dev/) - A clean and modern icon set that integrates well with Shadcn UI.
- **Date Handling:** [date-fns](https://date-fns.org/) - For formatting and manipulating dates in estimate history.
- **Exports:** [jspdf](https://github.com/parallax/jsPDF) and [docx](https://docx.js.org/) - For client-side generation of PDF and DOCX budget documents.
